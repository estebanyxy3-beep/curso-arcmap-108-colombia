📌 Sobre el curso

Una ruta práctica para aprender ArcMap con datos reales

Este curso está diseñado para personas que inician desde cero en SIG y necesitan trabajar con información territorial de Colombia. A través de ejercicios guiados, aprenderás a organizar datos, preparar capas, procesar información vectorial y ráster, calcular NDVI y exportar mapas profesionales.

🇨🇴

Datos reales de Colombia

Uso de fuentes como IGAC, DANE, IDEAM, Datos Abiertos Colombia y OpenStreetMap.

🛰️

Imágenes satelitales

Trabajo con Landsat 8/9 y Sentinel-2 para composiciones RGB y análisis territorial.

🌿

NDVI paso a paso

Cálculo e interpretación del índice de vegetación usando Raster Calculator y Float().

📄

Proyecto final

Entrega final con 7 mapas en PDF y un informe de interpretación territorial.

✅ Lo que aprenderás

Competencias prácticas al finalizar

Al completar el curso podrás crear un flujo de trabajo completo en ArcMap, desde la descarga de datos hasta la presentación cartográfica final.

  • ✓ Descargar datos geográficos oficiales de Colombia
  • ✓ Proyectar capas en Magna-SIRGAS / CTM12
  • ✓ Crear mapas base con simbología y etiquetas
  • ✓ Usar herramientas de geoprocesamiento como Clip, Buffer e Intersect
  • ✓ Generar hillshade, pendiente y curvas de nivel desde un DEM
  • ✓ Crear composiciones RGB en color natural y falso color
  • ✓ Calcular e interpretar el NDVI
  • ✓ Diseñar mapas profesionales en Layout View
  • ✓ Exportar mapas en PDF a 300 dpi
  • ✓ Unir tablas de Excel o CSV a datos geográficos
  • ✓ Digitalizar y editar información vectorial
  • ✓ Presentar un análisis territorial completo

🏁 Proyecto final

Proyecto Final — Análisis Territorial de un Municipio Colombiano

El proyecto integra cartografía base, análisis vectorial, procesamiento de DEM, composiciones satelitales y NDVI para construir una lectura territorial básica de un municipio colombiano.

  • 📄 Mapa base municipal
  • 📄 Mapa de vías, ríos y centros poblados
  • 📄 Mapa de hillshade y curvas de nivel
  • 📄 Mapa de pendiente
  • 📄 Composición RGB color natural
  • 📄 Composición RGB falso color infrarrojo
  • 📄 Mapa de NDVI clasificado
  • 📄 Informe escrito de interpretación
