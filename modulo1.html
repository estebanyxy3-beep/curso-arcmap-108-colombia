🧠 Conceptos fundamentales

Antes de abrir ArcMap, entiende la lógica de los datos

🌎

¿Qué es un SIG?

Un Sistema de Información Geográfica permite capturar, organizar, analizar y representar información con ubicación. Ayuda a responder preguntas como: ¿dónde está?, ¿qué hay cerca?, ¿qué relación existe entre elementos del territorio?

🖥️

¿Qué es ArcMap 10.8?

ArcMap es el programa central de ArcGIS Desktop. Permite cargar capas, consultar tablas, editar información, analizar datos y diseñar mapas finales para impresión o exportación.

🇨🇴

Ejemplos en Colombia

Puedes trabajar con municipios, vías, ríos, escuelas, zonas de riesgo, imágenes satelitales, modelos de elevación y datos estadísticos del DANE.

📐 Datos vectoriales

Representan elementos con geometría precisa. Se usan para objetos delimitados o identificables.

  • Puntos: escuelas, hospitales, estaciones.
  • Líneas: vías, ríos, redes.
  • Polígonos: municipios, predios, áreas de estudio.

🛰️ Datos ráster

Representan el territorio como una matriz de píxeles. Cada píxel tiene un valor numérico.

  • DEM o Modelo Digital de Elevación.
  • Imágenes Landsat y Sentinel-2.
  • NDVI, pendiente, temperatura o coberturas.
📁

Shapefile

Es un formato vectorial compuesto por varios archivos. Los más importantes son .shp, .dbf, .shx y .prj. Nunca deben separarse.

📊

Tabla de atributos

Es la tabla que contiene la información descriptiva de cada elemento del mapa. Cada fila representa un objeto y cada columna un campo.

🗂️

Proyecto MXD

El archivo .mxd guarda la organización del mapa, simbología y capas, pero no almacena los datos. Solo guarda referencias a sus ubicaciones.

🗂️ Organización del proyecto

La carpeta bien organizada evita errores durante todo el curso

Antes de cargar datos en ArcMap, crea una estructura de carpetas clara. Esta práctica evita rutas rotas, pérdidas de archivos y confusiones al exportar resultados.

⚠️ Regla de oro

No uses tildes, ñ, espacios ni caracteres especiales en nombres de carpetas o archivos. Usa guion bajo bajo _ y rutas cortas.

C:/SIG/Curso_ArcMap_108/ ├── 01_Vectorial ├── 02_Raster ├── 03_DEM ├── 04_Imagenes_Satelitales ├── 05_Resultados ├── 06_Mapas_PDF ├── 07_Proyecto_MXD └── 08_Tablas

🧪 Ejercicio práctico

Crear el primer proyecto en ArcMap 10.8

Sigue estos pasos en orden para configurar correctamente tu entorno de trabajo.

Crear la carpeta del curso

Crea la carpeta C:/SIG/Curso_ArcMap_108/ y dentro agrega las subcarpetas recomendadas.

Abrir ArcMap

Inicia ArcMap 10.8 y selecciona la opción para crear un mapa en blanco.

Conectar la carpeta en ArcCatalog

Abre ArcCatalog, usa Connect Folder y conecta la carpeta principal del curso.

Guardar el proyecto MXD

Guarda el proyecto como Curso_ArcMap_108.mxd dentro de la carpeta 07_Proyecto_MXD.

Explorar las vistas de trabajo

Alterna entre Data View y Layout View para reconocer la diferencia entre análisis y diseño final.

Activar Spatial Analyst

Ve a Customize → Extensions y verifica que Spatial Analyst esté activada.

Guardar nuevamente

Guarda el proyecto otra vez para conservar la conexión de carpetas y la configuración inicial.

🎯 Producto del ejercicio

Al terminar, debes tener una carpeta organizada con subcarpetas y un primer archivo .mxd guardado correctamente.

⚠️ Errores comunes

Evita estos problemas desde el primer módulo

⚠️
Guardar el MXD en el escritorio.

Solución: guarda siempre dentro de la carpeta del proyecto, en 07_Proyecto_MXD.

⚠️
Usar tildes, ñ o espacios.

Solución: usa nombres cortos, sin caracteres especiales y con guion bajo.

⚠️
Separar los archivos del shapefile.

Solución: mantén juntos los archivos .shp, .dbf, .shx y .prj.

⚠️
Mover datos después de crear el MXD.

Solución: no cambies las carpetas de ubicación después de cargar capas al proyecto.

⚠️
Confundir ArcMap con ArcGIS Pro.

Solución: recuerda que este curso está enfocado en ArcMap 10.8 de ArcGIS Desktop.

⚠️
No activar Spatial Analyst.

Solución: verifica la extensión desde Customize → Extensions antes de trabajar con ráster, DEM o NDVI.
