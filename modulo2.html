🔎 Términos clave explicados

Conceptos que debes dominar para descargar datos

En este módulo no solo se descargan archivos. También se aprende a reconocer qué tipo de dato se necesita, para qué sirve y cómo debe organizarse.

🏛️

Fuente oficial

Es una entidad o plataforma reconocida que publica datos confiables. En Colombia, ejemplos importantes son IGAC, DANE, IDEAM y Datos Abiertos Colombia.

🗺️

IGAC

El Instituto Geográfico Agustín Codazzi produce y administra información cartográfica, geográfica, catastral y geodésica del país. Es una fuente clave para límites, cartografía base y referencia espacial.

📊

DANE

El Departamento Administrativo Nacional de Estadística publica información estadística y geoestadística. Sus tablas pueden usarse para enriquecer capas mediante Join.

🌊

IDEAM

El Instituto de Hidrología, Meteorología y Estudios Ambientales aporta información ambiental, climática, hidrológica y de coberturas. Es útil para ríos, cuencas y análisis ambiental.

🌐

OpenStreetMap

Es una base de datos geográfica colaborativa mundial. Puede complementar datos oficiales con vías, edificaciones, sitios de interés y otros elementos.

📐

Dato vectorial

Representa objetos con puntos, líneas o polígonos. En este módulo se descargan como vectoriales los límites municipales, vías, ríos y centros poblados.

🛰️

Dato ráster

Representa información mediante píxeles. Cada píxel almacena un valor. En este curso se usan ráster como DEM, Landsat, Sentinel y NDVI.

⛰️

DEM

Significa Modelo Digital de Elevación. Es un ráster donde cada píxel guarda un valor de altura del terreno. Sirve para generar hillshade, pendiente y curvas de nivel.

🛰️

Landsat

Es una misión satelital que captura imágenes de la superficie terrestre. Sus bandas permiten crear color natural, falso color y calcular índices como NDVI.

🛰️

Sentinel-2

Es una misión satelital del programa Copernicus. Se usa mucho para análisis de vegetación, agua, coberturas y cambios del territorio.

🌍

EarthExplorer

Es una plataforma del USGS para buscar y descargar datos satelitales y modelos de elevación, como Landsat y SRTM.

⛰️

SRTM

Es una fuente global de modelos de elevación. En el curso se usa como DEM de aproximadamente 30 metros para análisis de relieve.

🌈

Banda satelital

Es una capa de una imagen satelital que registra una parte específica del espectro electromagnético. Por ejemplo, Landsat tiene bandas para azul, verde, rojo, infrarrojo cercano y otras.

☁️

Nubosidad

Es la presencia de nubes en una imagen satelital. Una escena con muchas nubes puede impedir ver el suelo, por eso se buscan imágenes con bajo porcentaje de nubosidad.

🧾

Metadatos

Son los datos que describen a otros datos: fuente, fecha, resolución, sistema de coordenadas, escala, sensor y condiciones de captura. Ayudan a saber si un archivo sirve para el análisis.

📦

ZIP

Es un archivo comprimido. Muchos portales entregan datos en ZIP. Antes de usarlo en ArcMap debes descomprimirlo y revisar qué archivos contiene.

📁

SHP

Es un formato vectorial. Debe conservar juntos sus archivos asociados. Si falta alguno, la capa puede no abrir o perder información.

📊

CSV

Es una tabla de texto separada por comas. En SIG se usa para unir datos estadísticos a capas geográficas, siempre que exista un campo clave común.

🎯 Objetivo

Preparar los datos base del curso

Al finalizar este módulo, tendrás una carpeta organizada con datos vectoriales, tablas, DEM e imágenes satelitales listos para cargar en ArcMap.

Idea clave

Un buen análisis en SIG empieza con una buena descarga de datos. Si descargas mal, nombras mal o guardas mal, el resto del proceso se complica.

📌

Qué debes lograr

  • Reconocer las fuentes de información geográfica.
  • Descargar datos pertinentes para el área de estudio.
  • Diferenciar vectorial y ráster.
  • Organizar y revisar los archivos antes de usarlos.

🧩 Tipos de datos

¿Qué son los datos vectoriales y qué son los datos ráster?

Antes de descargar, primero debes saber qué tipo de dato necesitas.

📐 Datos vectoriales

Representan objetos con geometrías exactas: puntos, líneas y polígonos.

  • Puntos: escuelas, hospitales, estaciones.
  • Líneas: vías, ríos, redes.
  • Polígonos: municipios, predios, áreas protegidas.

🛰️ Datos ráster

Representan el territorio como una malla de píxeles. Cada celda tiene un valor.

  • DEM: altura del terreno.
  • Landsat y Sentinel: reflectancia espectral.
  • NDVI: vigor de la vegetación.

Ejemplo de dato vectorial

Los puntos, líneas y polígonos representan objetos definidos del territorio.

Ejemplo de dato ráster

Una imagen satelital es una malla de píxeles que registra información del territorio.

Recuerda

Si el dato representa objetos concretos del terreno, normalmente es vectorial. Si parece una imagen o una superficie continua, normalmente es ráster.

🧾 Formatos

Formatos que encontrarás durante las descargas

📁

SHP

Formato vectorial clásico de ArcMap. Sus archivos deben permanecer juntos.

🗂️

GDB

Contenedor de Esri que puede almacenar múltiples capas.

🛰️

TIF

Formato ráster habitual para DEM o imágenes satelitales.

📊

CSV

Tabla de datos que después puede unirse a una capa mediante Join.

🗂️ Organización

Cómo guardar todo correctamente

La organización de carpetas te permitirá encontrar los datos rápido y evitar errores con las rutas en ArcMap.

Reglas básicas

No uses tildes, ñ ni espacios. Descomprime siempre los archivos ZIP antes de cargarlos. Conserva una copia original de las descargas.

🗺️ Descarga vectorial

Secuencia para descargar capas base

Estas capas se usan para construir el mapa base y el área de estudio.

Entrar al portal cartográfico.

Abre Colombia en Mapas o la fuente cartográfica oficial disponible.

Buscar el área de estudio.

Ubica el municipio o departamento que vas a trabajar.

Descargar límites administrativos.

Busca departamento, municipios o distritos según el análisis que necesites.

Descargar vías, ríos y centros poblados.

Estas capas servirán como base territorial para el proyecto.

Descargar tablas si es necesario.

Las tablas CSV serán útiles en módulos posteriores para unir información.

Descomprimir y clasificar.

Guarda cada tema en su carpeta correspondiente.

🛰️ DEM y Landsat

Secuencia para descargar SRTM y Landsat

Estos datos ráster se usarán después en análisis de relieve, composición RGB y NDVI.

⛰️

DEM SRTM

Ingresar a EarthExplorer.

Inicia sesión con tu cuenta.

Delimitar el área de estudio.

Ubica el municipio mediante coordenadas o polígono.

Seleccionar Digital Elevation → SRTM.

Escoge SRTM 1 Arc-Second Global.

Descargar la tesela correcta.

Verifica que cubra completamente el área de estudio.

🛰️

Landsat 8/9

Elegir Landsat 8-9 OLI/TIRS.

Es útil para color natural, falso color y NDVI.

Filtrar por fecha y nubosidad.

Busca escenas limpias, idealmente con baja nubosidad.

Revisar la vista previa.

Confirma que el municipio esté cubierto.

Descargar y guardar.

Organiza la escena en la carpeta de imágenes satelitales.

⚠️ Errores comunes

Evita estos errores

⚠️
No descomprimir los ZIP.

Primero descomprime, luego organiza y después carga.

⚠️
Descargar imágenes con muchas nubes.

Revisa siempre la vista previa antes de bajar una escena.

⚠️
Guardar todo mezclado.

Clasifica vectoriales, tablas, DEM e imágenes por carpetas.

⚠️
Olvidar registrar la fuente.

Anota de qué portal descargaste el dato.
