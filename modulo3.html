🔎 Términos clave explicados

Conceptos necesarios para entender coordenadas

Estos términos son la base del módulo. Si los comprendes, será más fácil revisar, corregir y proyectar capas en ArcMap.

📍

Coordenada

Es un valor o conjunto de valores que permite ubicar un punto en el espacio. En SIG, una coordenada indica dónde está un elemento sobre la superficie terrestre.

🧭

Sistema de coordenadas

Es el conjunto de reglas que define cómo se ubican los datos en el mapa. Le dice a ArcMap cómo interpretar los valores de posición de cada capa.

🌐

Coordenadas geográficas

Son coordenadas expresadas en latitud y longitud. Sus unidades son grados. Son comunes en GPS, imágenes satelitales y plataformas web.

📏

Coordenadas proyectadas

Son coordenadas expresadas en una superficie plana. Normalmente usan metros, por eso son adecuadas para medir áreas, distancias y hacer buffers.

↕️

Latitud

Indica qué tan al norte o al sur está un punto respecto al Ecuador. Se expresa en grados y puede tener valores positivos o negativos.

↔️

Longitud

Indica qué tan al este o al oeste está un punto respecto al meridiano de Greenwich. También se expresa en grados.

🔢

Grados decimales

Son una forma de escribir latitud y longitud usando números con decimales. Por ejemplo: 7.8890, -76.6340.

📐

X / Y

En coordenadas proyectadas, X normalmente representa el valor horizontal o Este, y Y representa el valor vertical o Norte.

🌎

Datum

Es la referencia matemática de la Tierra usada para calcular posiciones. Sirve como base para que las coordenadas se ajusten al planeta real.

🗺️

Proyección cartográfica

Es el método que transforma la superficie curva de la Tierra en un plano. Permite trabajar en metros, pero siempre genera algún tipo de deformación.

🌍

WGS84

Es un sistema de referencia global usado por GPS, imágenes satelitales y muchas plataformas web. Normalmente se trabaja en grados de latitud y longitud.

🇨🇴

MAGNA-SIRGAS

Es la referencia geodésica moderna de Colombia. Muchos datos oficiales del país se apoyan en este sistema para mantener coherencia espacial.

📏

MAGNA-SIRGAS / Origen-Nacional

Es un sistema proyectado usado en Colombia para trabajar en metros. Es muy útil para análisis espacial, medición de áreas, distancias y buffers.

🏷️

EPSG

Es un código que identifica sistemas de coordenadas. Por ejemplo, un sistema puede tener un número EPSG que ayuda a buscarlo correctamente en ArcMap u otros programas SIG.

🧾

Spatial Reference

Es la referencia espacial de una capa. En ArcMap se revisa en Layer Properties → Source. Allí puedes ver si la capa está en WGS84, MAGNA-SIRGAS u otro sistema.

📌

Define Projection

Es una herramienta que asigna o corrige la etiqueta del sistema de coordenadas de una capa. No transforma los valores de coordenadas.

🔁

Project

Es una herramienta que transforma una capa de un sistema de coordenadas a otro. Crea una nueva capa de salida con coordenadas convertidas.

🗺️

Data Frame

Es el marco de datos del mapa. También tiene un sistema de coordenadas que controla cómo se visualizan las capas dentro del proyecto.

🎯 Objetivo

Hacer que las capas estén bien ubicadas y hablen el mismo idioma espacial

Al finalizar, podrás revisar el sistema de coordenadas de una capa, identificar si está en grados o metros y proyectarla adecuadamente para seguir con el curso.

Idea clave

Una capa puede verse bonita en pantalla, pero si su sistema de coordenadas está mal, los análisis, distancias, áreas y recortes pueden salir incorrectos.

📌

Qué debes lograr

  • Diferenciar coordenadas geográficas y proyectadas.
  • Entender qué es un datum y qué es una proyección.
  • Reconocer sistemas usados en Colombia.
  • Aplicar Define Projection y Project correctamente.
❓ ¿Por qué importa?

Sin coordenadas correctas, las capas no coinciden

Si una capa está en grados y otra en metros, o si ambas tienen referencias diferentes, pueden aparecer desplazadas o dar resultados erróneos en análisis espaciales.

Ejemplo sencillo

Si haces un buffer de “100” en una capa geográfica, ese 100 no serán metros. Por eso primero debes revisar y proyectar correctamente la capa.

Cuando las capas comparten la misma referencia espacial, coinciden y permiten análisis correctos.

🧭 Tipos de coordenadas

Coordenadas geográficas y coordenadas proyectadas

Esta diferencia es una de las ideas más importantes del módulo.

🌐 Coordenadas geográficas

Ubican puntos con latitud y longitud. Sus unidades son grados.

Ejemplo:
Latitud 7.889° N
Longitud -76.634° W
  • Usan grados decimales.
  • Son comunes en GPS y plataformas web.
  • Sirven para ubicar, pero no son las mejores para medir.

📏 Coordenadas proyectadas

Ubican puntos en un plano. Sus unidades suelen ser metros.

Ejemplo:
Este 4.850.000 m
Norte 2.100.000 m
  • Usan metros.
  • Permiten calcular áreas, distancias y buffers.
  • Son ideales para análisis espacial.

Las geográficas se expresan en grados; las proyectadas en metros.

🌎 Datum

El datum es la referencia de la Tierra

El datum define el modelo matemático de la Tierra que usa el sistema de coordenadas. Si dos capas usan datum distintos, pueden no coincidir exactamente.

Forma sencilla de entenderlo

Piensa en el datum como la base o referencia desde la cual se “amarra” la ubicación del dato sobre el planeta.

Un mismo punto puede variar ligeramente si se calcula con referencias distintas.

🗺️ Proyección cartográfica

Proyectar es pasar de la superficie curva a un plano

Las proyecciones cartográficas permiten representar la Tierra en una superficie plana, aunque siempre deforman algo.

📐

Siempre deforma algo

Puede alterar áreas, formas, distancias o direcciones.

📏

Permite medir

Hace posible calcular distancias y áreas con unidades lineales como metros.

🇨🇴

Debe elegirse bien

En Colombia conviene usar sistemas apropiados al territorio nacional.

🇨🇴 Sistemas en Colombia

Sistemas que debes reconocer durante el curso

No todos los datos llegan con la misma referencia. Por eso debes aprender a identificarlos.

🌐

WGS84

Sistema muy común en GPS, plataformas web e imágenes satelitales. Normalmente aparece en coordenadas geográficas.

📍

MAGNA-SIRGAS

Es la referencia geodésica moderna de Colombia. Muchos datos oficiales del país la usan.

📏

MAGNA-SIRGAS / Origen-Nacional

Sistema proyectado usado para cartografía oficial y análisis en metros dentro de Colombia.

Recuerda

WGS84 es muy común para ubicar. MAGNA-SIRGAS / Origen-Nacional es muy útil para medir y analizar en Colombia.

🔁 Herramientas críticas

Define Projection y Project no significan lo mismo

Esta diferencia es fundamental y debes dominarla para no dañar tus capas.

📌 Define Projection

Se usa cuando la capa ya tiene coordenadas correctas, pero no tiene definido el sistema.

  • No transforma coordenadas.
  • Solo asigna la etiqueta del sistema.
  • Se usa cuando conoces el sistema original.

🔁 Project

Se usa cuando deseas convertir una capa de un sistema a otro.

  • Sí transforma coordenadas.
  • Crea una nueva capa de salida.
  • Se usa para pasar de grados a metros, por ejemplo.

Define Projection etiqueta la capa. Project la transforma a otro sistema.

⚠️ Errores comunes

Evita estos errores

⚠️
Confundir Define Projection con Project.

Una no transforma; la otra sí.

⚠️
Hacer análisis en grados.

Para medir o hacer buffers conviene trabajar en metros.

⚠️
Asignar un sistema al azar.

No inventes la proyección de una capa si no conoces su origen.

⚠️
No revisar todas las capas.

Debes comprobar una por una antes del análisis.
